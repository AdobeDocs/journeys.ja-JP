# lower {#lower}

パラメーターの小文字バージョンを返します。

## カテゴリ

文字列

## 関数の構文

`lower(<parameter>)`

## パラメーター

* 文字列

## 署名と戻り値の型

`lower(<string>)`

文字列を返します。

## 例

`lower("A")`

「a」を返します。
