# isEmpty {#isEmpty}

パラメーター内の文字列に文字がない場合はtrueを返します。

## カテゴリ

 バイト長文字列

## 関数の構文

`isEmpty(<parameters>)`

## パラメーター

* 文字列

## 署名と戻り値の型

`isEmpty(<string>)`

ブール値を返します。

## 例

`isEmpty("")`

trueを返します。

`isEmpty("Hello World")`

falseを返します。
