# contain {#contain}

2番目の引数文字列が最初の引数文字列に含まれているかどうかを確認します。

## カテゴリ

 バイト長文字列

## 関数の構文

`contain(<parameters>)`

## パラメーター

* 文字列

## 署名と戻り値の型

`contain(<string>,<string>)`

ブール値を返します。

## 例

`contain("rowing is great', "great")`

trueを返します。
