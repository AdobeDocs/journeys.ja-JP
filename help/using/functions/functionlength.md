# length {#length}

パラメーター内の文字列式ーの文字数を返します。

## カテゴリ

 バイト長文字列

## 関数の構文

`length(<parameters>)`

## パラメーター

* 文字列

## 署名と戻り値の型

`length(<string>)`

整数を返します。

## 例

`length("Hello World")`

11を返します。
