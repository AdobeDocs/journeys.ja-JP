# round {#round}

引数に最も近い整数値を返します。この値は、正の無限大に丸められます。

## カテゴリ

数学

## 関数の構文

`round(<parameters>)`

## パラメーター

* decimal
* integer

## 署名と戻り値の型

`round(<decimal>)`

`round(<integer>)`

整数を返します。

## 例

`round(3.14)`

3を返します。

`round(3.54)`

4を返します。

`round(-3.14)`

-3を返します。

`round(3)`

3を返します。
