# toTimeZone {#toTimeZone}

文字列値をタイムゾーンに変換します。

## カテゴリ

コンバージョン

## 関数の構文

`toTimeZone(<parameter>)`

## パラメーター

| パラメーター | 説明 |
|--- |--- |
| 文字列 | 文字列値には、ゾーンIDが含まれている必要があります。 フィールド参照または式を指定できます |

## 署名と戻り値の型

`toTimeZone(<string>)`

タイムゾーンを返します。

## 例

`toTimeZone("UTC")`

UTCを返します。
