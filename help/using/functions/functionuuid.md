# uuid {#uuid}

ランダムなUUID（ユニバーサル一意IDentifier）を生成します。

## カテゴリ

文字列

## 関数の構文

`uuid()`

## パラメーター

この関数にパラメーターは必要ありません。

## 署名と戻り値の型

`uuid()`

文字列を返します。

## 例

`uuid()`

戻り値は&quot;79e70b7f-8a85-400b-97a1-9f9826121553&quot;です。
