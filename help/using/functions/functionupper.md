# upper {#upper}

パラメーターの大文字を返します。

## カテゴリ

 バイト長文字列

## 関数の構文

`upper(<parameters>)`

## 署名と戻り値の型

`upper(<string>)`

文字列を返します。

## 例

`upper("b")`

&quot;B&quot;を返します。
