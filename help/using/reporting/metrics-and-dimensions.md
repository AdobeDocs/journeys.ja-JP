# 指標およびディメンション {#concept_rfj_wpt_52b}

>[!NOTE]
>
>配信データは、Adobe Campaign Standardを使用している場合にのみ入力されます。

動的レポートで使用可能なすべてのコンポーネントのリストとその定義は、こちらで確認できます。

次の表に、ジャーニーレポートで使用されるディメンションとその定義のリストを示します。

ディメンションと指標の互換性について詳しくは、[このページ](../assets/do-not-localize/dynamic_report_compatibility_journey.pdf)を参照してください。

## ジャーニーの寸法{#MBE_table_wk4_bnj_w2b}

次の表に、ジャーニーレポートで使用されるディメンション、その定義および数式のリストを示します。

| サイズ | 定義 |
|--- |--- |
| **アクション** | ジャーニーで使用されるすべてのアクション（**アクション名 — アクションラベル**）のリスト（例：プッシュ — チェックアウト確認、電子メール — 見返りの確認）。 |
| **データソース** | データをジャーニー(例：Adobe Experience Platform、予約システム)に拡張するために使用するデータソース（**データソース名**）のリスト。 |
| **[!UICONTROL イベント]** | ジャーニーで使用されるすべてのイベント(**イベント名 —イベントラベル**)のリスト(例：Geometrixxイベント-Geometrixxチェックアウト)。 |
| **フィールドグループ** | プロファイルフィールドグループ、Geometrixx予約システムなどのジャーニーにデータを拡張するために使用されるフィールドグループ（**フィールドグループ名**）のリスト。 |
| **道のり** | テストモードおよびライブモードでの各ジャーニー(**ジャーニー名**)のリスト（買い物かごの放棄、ホテルの予約通知など）。 |
| **ジャーニー版** | ジャーニーの公開バージョン(**ジャーニー名+バージョン番号**)のリスト（買い物かごの放棄v1、ホテルの予約通知v2など）。 |
| **オーケストレーション** | ジャーニーで定義および使用されるすべてのオーケストレーションアクティビティ(**Condition, End, Wait**)のリスト。 |

## 配信の寸法{#delivery-dimensions}

次の表に、ジャーニーレポートで使用される配信ディメンション、その定義および数式のリストを示します。

| ディメンション | 定義 |
|--- |--- |
| **ブラウザー** | メッセージが開かれた、またはクリックされたブラウザー。 |
| **配信名** | 配信のラベルとID。 |
| **デバイス** | 電子メール/SMS/プッシュ通知が開かれた、表示された、またはクリックされたデバイス。 |
| **メッセージタイプ** | 電子メール、SMS、プッシュ通知、アプリ内通知など、配信に使用されるチャネル。 |
| **モバイルアプリ名** | モバイルアプリケーションの名前 |
| **プラットフォーム** | メッセージが開かれた、表示された、またはクリックされたデバイスのプラットフォーム。 |
| **[!UICONTROL プッシュプラットフォーム]** | プッシュ通知が開かれたデバイスのプラットフォーム（iOS、Androidなど）。 |
| **受信者領域** | 電子メールを開くために使用するドメイン。 |
| **トラッキング URL** | ユーザーがメッセージからクリックしたURL。 |
| **追跡URLカテゴリ** | 追跡URLに割り当てられたカテゴリ。 |
| **追跡URLラベル** | URLに付与されるラベル(ミラーページなど)。お問い合わせいただくか、開いてください。 |
| **バリアント** | A/Bテストの場合の電子メールのバリアントです。 |

## ジャーニー指標{#MBE_p_p22_c4j_w2b}

次の表に、ジャーニーレポートで使用される指標のリスト、その定義と数式を示します。

| 指標 | 定義 |
|--- |---|
| **完了** | ジャーニーが正常に終了した個人の合計数です。 |
| **完了率** | ジャーニーに入った個人の合計数と比較して、通常はジャーニーを終了した個人の合計数です。 |
| **現在** | 現在ジャーニーにいる人の合計数、つまり入口から退出した人を引いた人数、エラーおよびタイムアウト。 |
| **現在の比率** | ジャーニーに入った個人の数と比較した、現在ジャーニーにいる個人の合計数です。 |
| **エントリ済み** | ジャーニー内の個々のエントリの開始に発生したイベントの合計数です。 |
| **エラー** | ジャーニー中に発生したが、ジャーニーの成功を妨げなかったエラーの合計数です。 |
| **アクションでのエラー** | アクションで発生したエラーの合計数です。 |
| **エンリッチメントでのエラー** | データソース/フィールドグループの呼び出し時に、データエンリッチメントで発生したエラーの合計数です。 |
| **イベントでのエラー** | イベントに発生したエラーの合計数。 |
| **エラー率** | ジャーニー中に発生したエラーの総数で、ジャーニー内のエラーの合計数と比較したものです。 |
| **実行されたアクション** | ジャーニーに対して実行されたアクションの合計数。 |
| **実行されたエンリッチメント** | データソースを呼び出して特定のフィールドグループを取得し、実行したエンリッチメントの総数です。 |
| **実行されたイベント** | ジャーニーに対して実行されたアクションの合計数。 |
| **実行されたオーケストレーション** | ジャーニーに対して実行されたオーケストレーションオブジェクト（終了、待機、条件）の合計数です。 |
| **失敗** | 正常に実行されなかったジャーニーの合計数です。 |
| **失敗率** | 実行ジャーニーの数と比較して、正常に実行されなかったジャーニーの合計数です。 |

## 配信指標{#delivery-metrics}

次の表に、ジャーニーで使用される指標のリストを示します
レポート、定義および数式を作成できます。

| 指標 | 定義 |
|--- |--- |
| **ブロックリスト登録人数** | 電子メールをスパムまたは迷惑メールとして宣言した受信者の数です。 |
| **ブロックリスト率** | 送信されたメッセージと比較した、ブロックリスト上のメッセージの合計数。 |
| **バウンス+エラー** | 配信および自動返信処理中に発生したエラーの合計で、送信されたメッセージの合計数に関連して計算されたエラーの合計です。 |
| **直帰率+エラー率** | 送信されたメッセージと比較してバウンスしたメッセージの合計数です。 |
| **クリック** | 配信内でコンテンツがクリックされた回数。 |
| **クリックスルー率** | 配信されたメッセージの数と比較した、配信内のクリック数の合計です。 |
| **配信済み** | 送信されたメッセージの合計数に関連して、正常に送信されたメッセージの数。 |
| **配送率** | 送信されたメッセージと比較して、正常に配信されたメッセージの合計数です。 |
| **エラー** | ジャーニー中に発生したが、ジャーニーの成功を妨げなかったエラーの合計数です。 |
| **ハードバウンス** | 誤った電子メールアドレスなど、永続的なエラーの合計数。 |
| **直帰率** | 永続的なエラーが原因で失敗した配信の合計数で、送信されたメッセージと比較したものです。 |
| **ミラーページ** | ミラーページリンクをクリックした受信者の数。 |
| **ミラーページ率** | 配信されたメッセージの合計数と比較した、ミラーページリンクのクリック数の合計です。 |
| **開封数** | 配信でメッセージが開かれた回数。 |
| **開封率** | 配信されたメッセージの数と比較した、開かれたメッセージの合計数です。 |
| **強制隔離** | アドレスの強制隔離につながった、バウンスしたメッセージの数です。 |
| **強制隔離率** | 送信されたメッセージと比較した強制隔離の合計数。 |
| **却下** | SMTPサーバーによってスパムとして分類されたメッセージの数です。 |
| **却下料金** | 拒否済みとマークされたメッセージの合計数で、送信されたメッセージと比較します。 |
| **処理済み/送信済み** | 配信の送信の合計数。 |
| **ソフトバウンス** | 完全な受信トレイなどの一時エラーの合計数です。 |
| **軟直帰率** | 一時的な理由により、送信された配信と比較して失敗したメッセージの合計数です。 |
| **ユニーククリック数** | 配信内のコンテンツをクリックした受信者の数です。 |
| **個別オープン** | 配信を開いた受信者の数です。 |
| **購読解除済み** | 購読解除リンクのクリック数。 |
| **登録解除率** | 配信されたメッセージと比較した、受信者別の購読解除の合計数です。 |
