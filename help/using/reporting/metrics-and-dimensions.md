# 指標およびディメンション {#concept_rfj_wpt_52b}

>[!NOTE]
>
>配信データは、Adobe Campaign Standardがある場合にのみ入力されます。

動的レポートで使用できるすべてのコンポーネントのリストとその定義は、こちらにあります。

次の表に、ジャーニーレポートで使用されるディメンションとその定義を示します。

ディメンションと指標の互換性について詳しくは、[このページ](../assets/do-not-localize/dynamic_report_compatibility_journey.pdf)を参照してください。

## ジャーニーディメンション {#MBE_table_wk4_bnj_w2b}

次の表に、ジャーニーレポートで使用されるディメンションのリストと、その定義および数式を示します。

| サイズ | 定義 |
|--- |--- |
| **アクション** | ジャーニーで使用されるすべてのアクション（**アクション名 — アクションラベル**）のリスト（例：プッシュ — 確認、Eメール — 報酬の正確性を確認）。 |
| **データソース** | ジャーニーでデータのエンリッチメントに使用するデータソース（**データソース名**）のリスト(例：Adobe Experience Platform、予約システム)。 |
| **[!UICONTROL イベント]** | Geometrixxイベント —Geometrixxチェックアウトなど、ジャーニーで使用されるすべてのイベント（**イベント名 — イベントラベル**）のリストです。 |
| **フィールドグループ** | ジャーニーでGeometrixxをエンリッチメントするために使用されるフィールドグループ（**フィールドグループ名**）のリスト（例：プロファイルフィールドグループ、データ予約システム）。 |
| **道のり** | テストモードおよびライブモードの各ジャーニーのリスト（**ジャーニー名**）（例：買い物かごの放棄、ホテルの予約通知）。 |
| **ジャーニーバージョン** | ジャーニーの公開済みバージョン（**ジャーニー名+バージョン番号**）のリスト（例：買い物かごの放棄v1、ホテルの予約通知v2）。 |
| **オーケストレーション** | ジャーニーで定義および使用されるすべてのオーケストレーションアクティビティ（**条件、終了、待機**）のリスト。 |

## 配信ディメンション{#delivery-dimensions}

次の表に、ジャーニーレポートで使用される配信ディメンション、その定義および数式のリストを示します。

| ディメンション | 定義 |
|--- |--- |
| **ブラウザー** | メッセージが開封またはクリックされたブラウザー。 |
| **配信名** | 配信のラベルとID。 |
| **デバイス** | 電子メール/SMS/プッシュ通知が開封、表示、クリックされたデバイス。 |
| **メッセージタイプ** | Eメール、SMS、プッシュ通知、アプリ内通知など、配信に使用されるチャネル。 |
| **モバイルアプリ名** | モバイルアプリケーションの名前 |
| **プラットフォーム** | メッセージが開封、表示、クリックされたデバイスのプラットフォーム。 |
| **[!UICONTROL プッシュプラットフォーム]** | iOSやAndroidなど、プッシュ通知が開かれたデバイスのプラットフォーム。 |
| **受信者ドメイン** | Eメールの開封に使用するドメイン。 |
| **トラッキング URL** | ユーザーがメッセージからクリックしたURL。 |
| **トラッキングURLカテゴリ** | トラッキングURLに割り当てられたカテゴリ。 |
| **トラッキングURLラベル** | ミラーページなどのURLに付与されたラベルは、お問い合わせいただくか、開いてください。 |
| **バリアント** | A/Bテストの場合のEメールのバリアント。 |

## ジャーニー指標 {#MBE_p_p22_c4j_w2b}

次の表に、ジャーニーレポートで使用される指標のリスト、その定義および数式を示します。

| 指標 | 定義 |
|--- |---|
| **完了** | 通常のジャーニーを終了した個人の合計数。 |
| **完了率** | ジャーニーに入った個人の合計数に対する、通常はジャーニーを終了した個人の合計数。 |
| **現在** | ジャーニーに現在参加している人の合計数（入口にした人から退出した人、エラー、タイムアウトした人を引いた数）。 |
| **現在の比率** | ジャーニーに参加した個人の数に対する、現在ジャーニーに参加している個人の合計数。 |
| **入場済み** | ジャーニーの個々のエントリを開始するために発生したイベントの合計数。 |
| **エラー** | ジャーニー中に発生したが、ジャーニーの成功を妨げないエラーの合計数。 |
| **アクションでのエラー** | アクションに対して発生したエラーの合計数。 |
| **エンリッチメントでのエラー** | データソース/フィールドグループの呼び出し時にデータエンリッチメントで発生したエラーの合計数。 |
| **イベントでのエラー** | イベントで発生したエラーの合計数。 |
| **エラー率** | ジャーニー内のエラーの合計数に対する、ジャーニー中に発生したエラーの合計数。 |
| **実行されたアクション** | ジャーニーで実行されたアクションの合計数。 |
| **実行されたエンリッチメント** | データソースを呼び出して特定のフィールドグループを取得することで実行されたエンリッチメントの合計数。 |
| **実行されたイベント** | ジャーニーで実行されたアクションの合計数。 |
| **実行されたオーケストレーション** | ジャーニーに対して実行されたオーケストレーションオブジェクト（終了、待機、条件）の合計数。 |
| **失敗** | 正常に実行されなかったジャーニーの合計数。 |
| **失敗率** | 実行ジャーニーの数に対する、正常に実行されなかったジャーニーの合計数。 |

## 配信指標{#delivery-metrics}

次の表に、ジャーニーで使用される指標のリストを示します
レポート、定義および数式を作成できます。

| 指標 | 定義 |
|--- |--- |
| **ブロックリスト登録人数** | Eメールをスパムまたは迷惑メールとして宣言した受信者の数。 |
| **ブロックリスト率** | 送信されたメッセージに対すブロックリストる、送信時のメッセージの合計数。 |
| **バウンス+エラー** | 送信されたメッセージの合計数に関する、配信および自動返信処理の間に累積したエラーの合計。 |
| **バウンス+エラー率** | 送信されたメッセージに対してバウンスしたメッセージの合計数。 |
| **クリック** | 配信でコンテンツがクリックされた回数。 |
| **クリックスルー率** | 配信されたメッセージ数に対する、配信でのクリック総数。 |
| **配信済み** | 送信されたメッセージの合計数に関する、正常に送信されたメッセージ数。 |
| **配信率** | 送信されたメッセージと比較して、正常に配信されたメッセージの合計数。 |
| **エラー** | ジャーニー中に発生したが、ジャーニーの成功を妨げないエラーの合計数。 |
| **ハードバウンス** | 誤った電子メールアドレスなどの永続的なエラーの合計数。 |
| **ハードバウンス率** | 送信されたメッセージに対する、永続的なエラーが原因で失敗した配信の合計数。 |
| **ミラーページ** | ミラーページのリンクをクリックした受信者の数。 |
| **ミラーページレート** | 配信されたメッセージの合計数に対する、ミラーページリンクでのクリック総数。 |
| **開封数** | 配信でメッセージが開かれた回数。 |
| **開封率** | 配信されたメッセージの数に対する、開封されたメッセージの合計数。 |
| **強制隔離** | アドレスの強制隔離につながった、バウンスしたメッセージの数。 |
| **強制隔離率** | 送信されたメッセージと比較した強制隔離の合計数。 |
| **却下** | SMTPサーバーによってスパムと分類されたメッセージの数。 |
| **却下率** | 送信されたメッセージに対する、拒否済みとマークされたメッセージの合計数。 |
| **処理済み/送信済み** | 配信の送信の合計数。 |
| **ソフトバウンス** | 完全なインボックスなどの一時的なエラーの合計数。 |
| **ソフトバウンス率** | 一時的な理由により送信されたメッセージと比較して失敗した配信の合計数。 |
| **ユニーククリック数** | 配信内のコンテンツをクリックした受信者の数。 |
| **個別開封数** | 配信を開封した受信者の数。 |
| **購読解除済み** | 購読解除リンクのクリック数。 |
| **購読解除率** | 配信されたメッセージに対する、受信者別の購読解除の合計数。 |
