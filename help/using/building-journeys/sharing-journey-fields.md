# ジャーニーのフィールド {#sharing-journey-fields}

このミックスインは、ジャーニースキーマで（jearryStepEventとの関連で）使用されます。 これには、次のフィールドが含まれます。

## jurneryID

メインジャーニーのID。

タイプ：string

## jeurneyVersionID

ジャーニーバージョンのID。 このIDは、ジャーニーのIDを表します。

タイプ：string

## name

ジャーニーの名前。

タイプ：string

## 説明

ジャーニーの説明。

タイプ：string

## version

バージョン。`major`.`minor`で表します。

タイプ：string
