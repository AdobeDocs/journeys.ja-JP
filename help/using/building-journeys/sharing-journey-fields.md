# ジャーニーのフィールド {#sharing-journey-fields}

このmixinは、journeyStepEventと関係のあるジャーニースキーマで使用されます。 次のフィールドが含まれます。

## journeyID

メインジャーニーのID。

型：文字列

## journeyVersionID

ジャーニーのバージョンのID。 このIDは、ジャーニーのIDを表します。

型：文字列

## name

ジャーニーの名前。

型：文字列

## 説明

ジャーニーの説明。

型：文字列

## version

バージョン。`major`.`minor`で表されます。

型：文字列
